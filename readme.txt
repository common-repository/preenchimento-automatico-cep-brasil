Preenchimento automático dos campos de endereço a partir de um CEP

== Description ==

Preenchimento automático dos campos de endereço a partir de um CEP

Utilizamos os serviços tercerizados da API https://viacep.com.br, cujo termos de uso são encontrados no link https://viacep.com.br

== Installation ==

= For an automatic installation through WordPress: =

1. Select **Add New** from the WordPress **Plugins** menu in the admin area.
2. Search for **Preenchimento Automatico CEP Brasil**.
3. Click **Install Now**, then **Activate Plugin**.

= For manual installation via FTP: =

1. Upload the **preenchimento-automatico-cep** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin from the **Plugins** screen in your WordPress admin area.

= To upload the plugin through WordPress, instead of FTP: =

1. From the **Add New** plugins page in your WordPress admin area, select the **Upload Plugin** button.
2. Select the **preenchimento-automatico-cep-brasil.zip** file, click **Install Now** and **Activate Plugin**.
